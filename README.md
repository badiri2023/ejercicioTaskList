Aplicacion para gestion de tareas.
Iniciamos la aplicacion y tenemos 2 botones, uno para agregar nueva tarea y otro para modo edicion de tareas.
Las tareas que se visualizará el nombre de la tarea y a su lateral un boton para eliminar la tarea.
Pulsamos Nueva tarea:
Al pusar el boton para añadir nueva tarea, este desplegará un input para introducir la nueva tarea junto a un boton para aceptar, una vez pulsado aceptar, podremos visualizar la tarea.
Pusamos  Editar:
Al pulsar el boton de editar entraremos al modo edicion, motrandonos debajo del header que nos encontramos en el modo edicion y nos deshabilitará el boton de nueva tarea,
ya que estamos editando no añadiendo, este nos mostrará que todas las tareaas hasta ahora cambiaran el boton de borrar por uno de editar, que al pusarlo nos permitirá modificar 
el nombre de la tarea y guardar el cambio, una vez acabada la edicion, nos permitirá descativar el modo edicion pulsando el boton nuevamente.
